
# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers. It is built using modern web technologies, including Web Components, to create a modular and maintainable codebase.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The main stylesheet.
*   `main.js`: The main JavaScript file, which contains the logic for the lottery number generator.
*   `blueprint.md`: This file, which contains the project blueprint.

## Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers in a visually appealing way.
*   A button to generate new numbers.
